# How to install
### Manually:
1. Open Fpilot-Config.json, located in `%AppData%/Voidstar/FilePilot/`
2. Copy and paste the code below this line:
```
},
"Colors":
[
```
It'll look like this:
```
},
"Colors":
[
  "Eon Dark: Green":
    {
    "Clear": "262826",
    "Caption": "262826",
...
```

3. Save the file.
---
### Ready-to-use .json:
You can also download this ready-to-use file: https://github.com/ru-bem/EON-themes/blob/main/FilePilot/Fpilot-Config.json <br>
Just copy and paste it inside `%AppData%/Voidstar/FilePilot/`
